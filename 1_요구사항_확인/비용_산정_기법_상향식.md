# 1. LOC(원시 코드 라인 수, source lines of code) 기법
> - 각 기능의 원시 코드 라인 수의 비관치, 낙관치, 기대치를 측정하여 예측치를 구하고 이를 이용하여 비용을 산정하는 기법
> - 예측치 = (낙관치 + 4 * 기대치 + 비관치) / 6
> - **산정 공식**
>   - 노력(인월) = 개발 기간 * 투입 인원
>   - 개발 비용 = 노력(인월) * 인건비(단위 비용 ; 1인당 월 평균 인권비)  
>   - 개발 기간 = 노력(인월) / 투입 인원
>   - 생산성 = LOC / 노력(인월)
